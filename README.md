# Multi-Vendor E-Commerce Platform

A full-stack Multi-Vendor E-Commerce application built using React.js for frontend and FastAPI for backend.

## Features

### Authentication
- User Login
- JWT Token Authentication
- Role-based Access (Admin, Vendor, Customer)

### Product Management
- View Products
- Add Products
- Product Images
- Product Details

### Cart System
- Add to Cart
- Remove from Cart
- View Cart

### Order Management
- Place Orders
- View Orders

### Frontend
- React.js
- Axios API Integration
- Responsive UI

### Backend
- FastAPI
- SQLAlchemy
- JWT Authentication
- REST APIs

## Tech Stack

### Frontend
- React.js
- Axios
- React Router DOM

### Backend

## Installation

### Backend Setup

cd backend

pip install -r requirements.txt

uvicorn app.main:app --reload

### Frontend Setup

cd frontend

npm install

npm start

## Run Application

Frontend:
http://localhost:3000

Backend:
http://127.0.0.1:8000
